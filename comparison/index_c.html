About NDC Comparison

This dashboard allows you to compare transport commitments across countries and NDC generations.

How to use:

  • Select countries and generations: Use the dropdowns in each column to choose which country and generation to display
  • Choose a tab: View Mitigation or Adaptation targets and measures
  • Apply filters: Use the checkboxes below the tabs to filter by GHG/Non-GHG, A-S-I, or transport modes
  • Compare: Scroll down to see all three columns side by side
